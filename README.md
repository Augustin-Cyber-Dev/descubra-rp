## 📖 Sobre o Projeto
 
O **Descubra RP** é um site de guia turístico digital desenvolvido como projeto acadêmico do curso de Ciências da Computação na UNIP. O objetivo foi aplicar na prática os conhecimentos adquiridos ao longo do semestre, criando uma aplicação web real e funcional.
 
O site reúne as principais atrações, restaurantes e experiências da cidade de Ribeirão Preto, SP, apresentando tudo de forma moderna, responsiva e acessível.
 
---
 
## ✨ Funcionalidades
 
- 🏙️ Apresentação das principais atrações turísticas de Ribeirão Preto
- 🍽️ Guia de restaurantes e gastronomia local
- 📱 Layout totalmente responsivo para mobile e desktop
- 🧭 Navegação multi-páginas organizada por categorias
- ⭐ Seção de avaliações de usuários
- 🔗 Integração com redes sociais
